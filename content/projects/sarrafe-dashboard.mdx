The figma file of this one given recently to me as a technical case for a job application.

This is a fictional dashboard for a company named Sarrafe. The dashboard is designed to show the company's financial data, such as revenue, expenses, and profit. The dashboard also includes a section for recent transactions etc.

I put effort to make it responsive across various devices phone, tablet and desktop. Also added dark mode support to the dashboard.
